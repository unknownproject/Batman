[Home]                          - Activate Main Debug Menu
[PageUp]      /   [PageDown]    - Navigation [Main Debug Menu]
[DownArrow]   /   [UpArrow]     - Navigation [2nd Level Menu]
[Num+]        /   [Num-]        - Navigation [3rd Level Menu]
[RightArrow]  /   [LeftArrow]   - Navigation [4th Level Menu]
